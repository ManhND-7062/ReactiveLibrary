# ReactiveLibrary

A Reactive Programming Library for Lua

## Overview

ReactiveLibrary is a powerful and flexible reactive programming library designed for Lua.  It provides developers with tools to build reactive systems using functional programming principles, enabling you to compose and manage complex data flows with ease.

## Features

- **Observable Streams**: Create and manage observable data streams
- **Operators**: Rich set of reactive operators for data transformation
- **Subjects**: Multicast observables for multiple subscribers
- **Error Handling**: Comprehensive error handling and recovery mechanisms
- **Lua Compatible**: Built specifically for Lua with no external dependencies
